# deckgl-starter

![README02](img/README02.png)

Start deck.gl easily.  
- [deck.gl v8.8.14](https://maplibre.org)  
- [TypeScript v4.8.4](https://www.typescriptlang.org)
- [Vite v3.1.8](https://vitejs.dev)  
- node v18.1.0
- npm v8.8.0

<br>

## Usage

![README03](img/README03.png)

<br>

Install package
```bash
npm install
```

<br>

build
```bash
npm run build
```

<br>

dev
```bash
npm run dev
```
